## 📖 프로젝트 소개
RoT 프로젝트는 무선 네트워크를 통해 하드웨어를 제어하고 데이터를 저장할 수 있는 시스템을 구축한 것입니다. <br><br>
이를 통해 다양한 기능을 원격으로 제어하고 실시간으로 모니터링할 수 있습니다. <br><br>
간단한 구조를 통해 실생활에서 사용되는 IoT 시스템을 간접적으로 체험할 수 있도록 설계되었습니다.
